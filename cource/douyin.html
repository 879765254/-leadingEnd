Open Api

  • 帐号授权

    包含抖音帐号授权登录与授权绑定的能力

  • 用户管理

    包含用户的基础信息以及关系链相关数据的获取

  • 视频管理

    包含抖音、西瓜、头条视频的发布以及相关数据的获取能力

  • 互动管理

    包含私信收发以及评论管理的能力

  • 搜索管理

    包含通过关键词获取抖音视频及该视频下评论，并进行回复的能力

  • 数据开放服务

    包含用户授权后的公开用户数据，视频数据等能力

  • 企业号开放能力

    包含企业号意向用户、企业消息卡片、团购活动等的管理能力

  • 生活服务开放能力

    包含商铺/商品库/交易系统等能力的开放接入
小程序

  • 接入流程

    小程序对企业账号及个人账号开放，一次开发可在字节跳动系APP上同时运用

  • 流量入口

    文章、视频、直播等多种体裁可按内容需求和相关性挂载小程序，实现内容到服务的用户升级需求满足

  • API

    字节小程序开发框架提供了丰富的原生 API 和 开放能力 API，开发者可方便快捷地调用这些 API完成业务逻辑

  • 组件

    小程序框架为开发者提供了一系列基础组件，开发者可以通过组合这些基础组件进行业务开发

  • 开放能力

    字节小程序为满足开发者的业务诉求，将主要的业务能力开放出来，开发者的小程序快速完成与字节服务端的交互

  • 运营规范

    开发者根据文档中的规范自行开发小程序，经审核后即可上架，上架后的小程序可在丰富的场景中配置展示
